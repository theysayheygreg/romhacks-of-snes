# External Research

This folder is for research gathered from external assistants, web sweeps, and broad source aggregation passes.

Use it for:

- Gemini deep-research briefs
- curated link dumps that still need distillation
- cross-source comparison notes from external assistants

Do not treat files here as canonical by default.

Canonical workspace notes should live one level up in `snes/docs/` after they have been distilled, checked, and rewritten for this project.

Suggested naming:

- `gemini-deep-research-001.md`
- `gemini-deep-research-002.md`
- `video-source-digest-001.md`
